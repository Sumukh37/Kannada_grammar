ಸಂಧಿ


ಆಗಮ ಸಂಧಿ


ಯಕಾರಗಮ ಸಂಧಿ
ಸಂಧಿ ಕಾರ್ಯ ಮಾಡಿದಾಗ ಒಂದು ಅಕ್ಷರವು ಹೊಸದಾಗಿ ಬಂದು ಸೇರುವುದನ್ನು
ಆಗಮ ಸಂಧಿ ಎನ್ನುವರು.
ಆ, ಇ, ಈ, ಎ, ಏ, ಐ, ಓ ಸ್ವರಗಳ ಮುಂದೆ ಸ್ವರವು ಬಂದರೆ, ಸಂಧಿ ಪದದಲ್ಲಿ
"ಯ್" ವ್ಯಂಜನವು ಹೊಸದಾಗಿ ಆಗಮವಾಗುತ್ತದೆ. ಇದಕ್ಕೆ ಯಕಾರಾಗಮ ಸಂಧಿ
ಎಂದು ಹೆಸರು.
ಉದಾ : ಕೆರೆ + ಅಲ್ಲಿ = ಕೆರೆಯಲ್ಲಿ
ಗಾಳಿ + ಅನ್ನು = ಗಾಳಿಯನ್ನು
ಮಳೆ + ಇಂದ = ಮಳೆಯಿಂದ
ಮೇ + ಇಸು = ಮೇಯಿಸು
ವಕಾರಗಮ ಸಂಧಿ
ಉ, ಊ, ಋ, ಓ, ಔ ಸ್ವರಗಳ ಮುಂದೆ ಸ್ವರವು ಬಂದರೆ ಸಂಧಿ ಪದದಲ್ಲಿ
"ವ್" ವ್ಯಂಜನವು ಹೊಸದಾಗಿ ಆಗಮವಾಗುತ್ತದೆ. ಇದಕ್ಕೆ ವಕಾರಾಗಮ ಸಂಧಿ
ಎಂದು ಹೆಸರು.
ಉದಾ : ಮಗು + ಇಗೆ = ಮಗುವಿಗೆ
ಹೂ + ಇದು = ಹೂವಿದು
ಗುರು + ಅನ್ನು = ಗುರುವನ್ನು
ಮಾತೃ + ಅನ್ನು = ಮಾತೃವನ್ನು
